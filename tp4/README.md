### Tangram

In this exercice we saw that to map a texture we first would need to define each point of the texture.
Each texture coordinate gets mapped to the corresponding vertex:

E.g.

```js
this.vertices = [
-0.5, -0.5, 0,	// Linked
    ...
];

this.texCoords = [
    0, 1,   // Linked
    ...
]
```
